# Project: base2048

**Stack:** Cloudflare Workers + Pages + D1 + KV | Base mainnet (8453) | ERC721 NFT + RevenuePool contract

**Domains:** cloudflare-workers, evm-base-chain, frontend-web3

---

### Key decisions made

- NFT ownership tracked in D1 `nft_owners` table, kept live by Alchemy Address Activity webhook
- Leaderboard in D1 `leaderboard` table, kept live by Alchemy ScoreCard webhook  
- Sessions stored in KV as `sess:{wallet_lowercase}` JSON
- Holder cache in KV as `hc:{wallet}` with 300s TTL, invalidated on Transfer webhook
- Primary RPC: `base-rpc.publicnode.com` | Fallback: `env.ALCHEMY_RPC`
- Free sessions: 20/month for NFT holders (`HOLDER_FREE_PER_MONTH = 20`)
- Game-over: always show overlay + "Use a Life" button, never auto-consume

### Contract selectors confirmed working
- `pendingFor(uint256)` on RevenuePool → `0x358d3097`
- `claim(uint256)` → `0x379607f5`
- `claimBatch(uint256[])` → `0x62abebce`
- `balanceOf(address)` on NFT → `0x70a08231`
